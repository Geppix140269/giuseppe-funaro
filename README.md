# Giuseppe Funaro — Personal CV Website

A single-page, animated personal CV / portfolio site.

**Live site:** _add your URL here once published_

## What it is
A self-contained `index.html` (no build step, no dependencies beyond a Google Fonts link).
Executive theme, scroll-reveal animations, animated impact counters, an interactive career
timeline, and language proficiency bars. Fully responsive.

## How to run locally
Just open `index.html` in any browser. That's it.

## How to publish on GitHub Pages
1. Create a new public repository on GitHub (suggested name: `giuseppe-funaro` or `cv`).
2. Upload `index.html` and this `README.md` (drag-and-drop in the GitHub web UI works).
3. Go to **Settings → Pages**, set **Source: Deploy from a branch**, branch `main`, folder `/ (root)`, Save.
4. Your site goes live at `https://<your-username>.github.io/<repo-name>/` within a minute or two.

## Custom domain (optional)
Buy a domain (e.g. `giuseppefunaro.com`), then in **Settings → Pages → Custom domain**
enter it and follow the DNS instructions. A `CNAME` file will be added automatically.

## Editing
All content lives in `index.html`. Text, roles and numbers are plain HTML you can edit directly.
